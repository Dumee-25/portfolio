Started learning ROS 2 today. Everything I've built so far has been
software that runs on my own machine and answers to me; ROS 2 is my first
real exposure to the robotics side — nodes, topics, quality-of-service
profiles, a middleware built around the idea that parts of a system can fail
or arrive late and the rest has to keep working anyway.

No project yet, just working through the tutorials and getting a feel for
how a robotics stack is put together. Recording the start date here mostly
so I can look back later and see how much of this was still confusing at the
one-month mark.
